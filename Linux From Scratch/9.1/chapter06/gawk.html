6.57. Gawk-5.0.1

The Gawk package contains programs for manipulating text files.

Approximate build time: 0.4 SBU
Required disk space: 47 MB

6.57.1. Installation of Gawk

First, ensure some unneeded files are not installed: 

sed -i 's/extras//' Makefile.in

Prepare Gawk for compilation: 

./configure --prefix=/usr

Compile the package: 

make

To test the results, issue: 

make check

Install the package: 

make install

If desired, install the documentation: 

mkdir -v /usr/share/doc/gawk-5.0.1
cp    -v doc/{awkforai.txt,*.{eps,pdf,jpg}} /usr/share/doc/gawk-5.0.1

6.57.2. Contents of Gawk

Installed programs: awk (link to gawk), gawk, and awk-5.0.1
Installed libraries: filefuncs.so, fnmatch.so, fork.so, inplace.so, intdiv.so, ordchr.so, readdir.so, readfile.so, revoutput.so, revtwoway.so, rwarray.so, and time.so (all in /usr/lib/gawk)
Installed directories: /usr/lib/gawk, /usr/libexec/awk, /usr/share/awk, and /usr/share/doc/gawk-5.0.1

Short Descriptions

awk

A link to gawk

gawk

A program for manipulating text files; it is the GNU implementation of awk
